## 🧪 测试步骤

### 步骤 1: 测试数据提交
1. 打开问卷页面
2. 按 `F12` 打开浏览器控制台
3. 填写并提交一份测试问卷
4. 查看控制台输出，应该看到：
   ```
   ========== 准备提交数据 ==========
   姓名: [您的名字]
   行业: [您的行业]
   最想学习的AI技能: [选择的技能]
   总得分: [分数]
   ✓ 方法1 请求已发送
   ✓ 方法2 请求已发送
   ```

### 步骤 2: 检查 Google Sheets
1. 打开您的 Google Sheets
2. 应该能看到新的一行数据
3. 检查数据是否正确：
   - 姓名是否正确
   - 行业是否正确
   - 总分是否正确
   - 想学习的AI技能是否正确

### 步骤 3: 检查 Google Apps Script 执行记录
1. 打开 Google Apps Script 编辑器
2. 点击 **执行** > **查看执行记录**
3. 查看最近的执行记录，应该看到：
   ```
   ========== 收到 POST 请求 ==========
   成功解析 parameter.data
   ========== 开始处理数据 ==========
   ✓ 数据已成功保存
   ```

## 🔧 快速修复

### 如果看到 "SPREADSHEET_ID 未配置" 错误：
1. 打开 Google Apps Script
2. 找到第 8 行：`const SPREADSHEET_ID = 'YOUR_SPREADSHEET_ID_HERE';`
3. 替换为您的 Google Sheets ID
4. 保存并重新部署

### 如果看到 "无法解析请求数据" 错误：
1. 检查 Web App URL 是否正确
2. 确认 Web App 已部署
3. 确认访问权限设置为"任何人"

### 如果数据保存到错误的列：
1. 检查 Google Sheets 表头顺序
2. 运行 `setupHeaders()` 函数重新创建表头
